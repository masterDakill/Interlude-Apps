## Quick Start Guide

### 1Ô∏è‚É£ Songs Library

**Adding Your First Song:**
1. Click on the "Songs" tab in the navigation
2. Click the "Add Song" button
3. Fill in the song details:
   - Title and Artist (required)
   - Musical information: Key, Tempo (BPM), Duration
   - Difficulty level: Easy, Medium, or Hard
   - Current status: Learning, Rehearsing, or Ready
   - Optional: Lyrics, Chords, and Notes
4. Click "Add Song"

**Tracking Practice Sessions:**
1. Find the song in your library
2. Click the "Log Practice" button
3. Enter:
   - Practice duration (in minutes)
   - Quality rating (1-5 stars)
   - Notes about what you worked on
4. View your practice history in the modal

**Managing Songs:**
- **Edit**: Click the pencil icon to modify song details
- **Delete**: Click the trash icon to remove a song
- **View Stats**: See total practice time and number of sessions

### 2Ô∏è‚É£ Setlist Planner

**Creating a Setlist:**
1. Go to the "Setlists" tab
2. Click "Create Setlist"
3. Enter a name (e.g., "Summer Tour 2024", "Wedding Set")
4. Add songs from your library:
   - Browse available songs
   - Click "Add" next to songs you want
   - Songs appear in the order added
5. Reorder songs using ‚ñ≤‚ñº buttons
6. See total duration automatically calculated
7. Click "Create Setlist"

**Editing Setlists:**
- Click the edit icon on any setlist
- Add/remove songs as needed
- Reorder by moving songs up or down
- Update setlist name or notes

**Best Practices:**
- Create different setlists for different venue types
- Consider song flow and energy levels
- Watch total duration for time constraints
- Add notes about transitions or special requirements

### 3Ô∏è‚É£ Show Manager

**Scheduling a Show:**
1. Navigate to the "Shows" tab
2. Click "Schedule Show"
3. Enter show details:
   - Show name
   - Venue name
   - Date and time
   - Choose a setlist (optional)
   - Set status: Planned, Confirmed, Completed, or Cancelled
   - Add any notes
4. Click "Schedule Show"

**Managing Shows:**
- Shows are automatically sorted into "Upcoming" and "Past"
- View stats: Total shows, upcoming, confirmed, completed
- Edit show details at any time
- Link different setlists as plans change
- Update status as show progresses

**Show Status Workflow:**
1. **Planned** - Initial booking
2. **Confirmed** - Venue confirmed, contract signed
3. **Completed** - Show finished
4. **Cancelled** - Show cancelled

### 4Ô∏è‚É£ Technical Rider

**Creating a Tech Sheet:**
1. Go to "Technical Rider" tab
2. Select a show from the dropdown
3. Fill in sections as needed:

**Stage Requirements:**
- Stage dimensions (width x depth in feet)
- Riser requirements
- Power needs

**Sound Requirements:**
- Number of monitor wedges needed
- Monitor mix preferences
- PA system requirements

**Channel List:**
- Click "Add Channel" for each input
- Specify:
  - Instrument/source
  - Microphone type preference
  - DI box requirement (checkbox)
  - 48V phantom power (checkbox)
  - Special notes
- Channels are auto-numbered
- Remove channels with trash icon

**Backline:**
- Click "Add Item" for each piece of gear
- Specify:
  - Item name (e.g., "Guitar Amp")
  - Quantity
  - Whether you're providing it (checkbox)
  - Notes about specifications
- Track what you bring vs. what venue provides

**Lighting:**
- Type of lighting preferred
- Special effects needed
- Color preferences
- General lighting notes

**Additional Notes:**
- Any other technical requirements
- Load-in times
- Special considerations
- Contact information

**Exporting Tech Sheets:**
- Click "Export" to download as a text file
- Click "Print" to print directly
- Send to venue/sound engineer before show

## üí° Pro Tips

### Song Organization
- Use consistent key notation (e.g., "C major" or "Cmaj")
- Keep tempo accurate for better planning
- Update status as you master songs
- Log practice regularly to track improvement

### Setlist Strategy
- Start strong with an attention-grabber
- Vary energy levels throughout
- Group songs in similar keys when possible
- Plan for encores
- Consider audience requests

### Show Planning
- Create tech sheets well in advance
- Update setlist based on venue/audience
- Keep notes about what worked at each venue
- Track completed shows for statistics

### Technical Sheets
- Be specific but flexible
- Include acceptable alternatives
- Update your standard rider as needs change
- Keep a "touring rider" template setlist
- Communicate early with venues

## üîÑ Workflow Example

**Planning a Wedding Gig:**

1. **Add Songs** (Songs Library)
   - Add all potential wedding songs
   - Mark difficulty and learning status
   - Practice and log sessions

2. **Create Setlist** (Setlist Planner)
   - Create "Wedding Set - Ceremony"
   - Create "Wedding Set - Reception"
   - Organize by moment (entrance, first dance, etc.)

3. **Schedule Show** (Show Manager)
   - Add venue and date
   - Link reception setlist
   - Mark as "Confirmed" when booked

4. **Prepare Tech Sheet** (Technical Rider)
   - Fill in channel list for your setup
   - Note if using DJ equipment
   - Specify ceremony wireless needs
   - Export and send to venue

5. **Day of Show**
   - Print setlists and tech sheet
   - Update show status to "Completed" after
   - Log any notes for future reference

## üÜò Troubleshooting

**Data Not Saving?**
- Check if cookies/localStorage enabled
- Ensure browser isn't in private/incognito mode
- Try refreshing the page

**Can't See My Songs in Setlist?**
- Make sure songs are saved in Songs Library first
- Refresh the page to sync data

**Tech Sheet Not Exporting?**
- Check browser allows downloads
- Try using Print instead
- Ensure a show is selected
